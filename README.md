# Recod 🎙️

**Recod** — это современное приложение для записи и транскрибации аудио на macOS, использующее локальную модель Whisper (WhisperKit).

## Возможности

- 🎤 **Запись** одним кликом из строки меню
- 🔊 **Запись системного звука** (стерео разделение: голос/система)
- ⚡ **Локальная транскрибация** (без интернета) с помощью WhisperKit
- ⏱️ **Детальная расшифровка**: Просмотр записи по сегментам с таймкодами (формат `0:00.0`)
- 📋 **Вставка текста без затирания буфера обмена** (сохранение картинок/файлов)
- ⌨️ **Глобальные хоткеи** (по умолчанию `Cmd+Shift+R`)
- 📄 **Авто-обновления** (через Sparkle)
- 🎨 **Современный дизайн** в стиле macOS (Glassmorphism)

## Установка

1. Перейдите в [Releases](https://github.com/McKrei/recod/releases)
2. Скачайте последнюю версию `Recod.zip`
3. Распакуйте и переместите `Recod.app` в `/Applications`
4. **Важно:** При первом запуске выполните команду в терминале, чтобы снять карантин:
   ```bash
   xattr -cr /Applications/Recod.app
   ```
   (Или разрешите запуск через Системные настройки → Конфиденциальность и безопасность)

## Разработка

Проект использует Swift 6 и SwiftUI.

### Требования
- macOS 15+ (Sequoia)
- Xcode 16+

### Запуск локально
```bash
make run
```

### Сборка релиза
Для создания `.app` бандла:
```bash
make app
```
Бандл появится в `build/Recod.app`.

### Выпуск обновления (CI/CD)
Автоматическое создание тега и запуск GitHub Actions:
```bash
make release
# или для мажорной версии:
make release MAJOR=2
```
