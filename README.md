
## Planned Features
- в настройки добавить возможность выбрать модель,
клик по ней переход на страницу со списоком моделей, сразу whisper имя, вес модели, размер, кнопка скачать, при нажатии на которую начинается загрузка модели, после загрузки появляется кнопка удалить, при нажатии на которую удаляется модель и освобождается место на диске. Нажатием на модель можно ее выбрать (будет использоваться для распознавания) и при этом она будет выделяться, а остальные модели будут неактивными.
